# clappr-airplay-plugin

> AirPlay support for Clappr

## Installing

For install dependencies, use npm or yarn:

```bash
$ npm install
```

## Developing

Start develop mode:

```bash
$ npm run start
```

Start watch server for exec based on file changes:

```bash
$ npm run watch
```

## Packing'n Release

```bash
$ npm run release
```
